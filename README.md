# README

This is the oh-my-zsh theme that I customized and use.

## Requirements
- zsh installed
- oh-my-zsh installed

## Installation
``` bash
git clone https://github.com/Ngibb/nolan-omzsh-theme.git
ln -s  ~/nolan-omzsh-theme/.nolan.zsh-theme ~/.oh-my-zsh/themes/ngibb.zsh-theme
```
